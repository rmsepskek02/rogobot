## 작성 양식

```
### [날짜] 실수 제목
- **상황**: 어떤 상황에서 발생했는가
- **실수**: 무엇을 잘못했는가
- **원인**: 왜 발생했는가
- **방지책**: 앞으로 어떻게 방지할 것인가
```
