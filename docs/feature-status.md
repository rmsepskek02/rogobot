## 2. 로아API (구버전) — 이전 필요

> 가장 큰 이전/리팩토링 작업 대상. 출처: `msgbot/Bots/로아API/로아API.js`  
> ⚠️ 공통 주의사항: 모든 기능에서 **엘릭서, 초월 관련 데이터는 제거**하여 이전할 것

| 기능 | 명령어 | 상태 | 비고 |
|------|--------|------|------|
| 배럭/부캐 목록 + 주급 계산 | `!배럭` `!부캐` `!ㅂㅋ` `!ㅂㄹ` `!원정대` `!ㅇㅈㄷ` | ✅ 이전 완료 | 주급 골드 계산 로직 포함 |
| 보석 정보 | `!보석 이름` | ✅ 이전 완료 | 멸화/홍염/겁화/작열 정렬 출력 |
| 스킬 정보 | `!스킬 이름` | ✅ 이전 완료 | 룬 장착 스킬 + 트라이포드 출력 |
| 장비 정보 | `!장비 이름` | ✅ 이전 완료 | 부위별 강화 출력 (엘릭서/초월 제거) |
| 악세 정보 (텍스트) | `!악세 이름` | ✅ 이전 완료 | 텍스트 출력 |
| 각인 상세 | `!각인 이름` | ✅ 이전 완료 | 각인명/등급/Lv 상세 출력 |
| 아크패시브 | `!앜패` `!ㅇㅍ` | ✅ 이전 완료 | 진화/깨달음/도약 포인트 상세 |
| 아크그리드 | `!아크` `!ㅇㅋ` | ✅ 이전 완료 | 코어/젬 효과 요약 |
| 내실 (수집품) | `!내실 이름` | ✅ 이전 완료 | 성향/수집품/스포 출력 |
| 낙원력 | `!낙원` `!낙원력` `!ㄴㅇ` `!ㄴㅇㄹ` | ✅ 이전 완료 | 원정대 캐릭터별 낙원력 계산 |
| 쌀 섬 일정 | `!섬` `!ㅅ` `!ㅆ` | ✅ 이전 완료 | 게임콘텐츠 API 기반 골드 섬 일정 |
| 유각 가격 | `!유각` `!ㅇㄱ` | ✅ 이전 완료 | 거래소 유물 각인 가격 조회 |
| 서폿 유각 가격 | `!서폿유각` `!ㅅㅍㅇㄱ` | ✅ 이전 완료 | 서포터용 유각 8종 가격 조회 |
| 에기르 재료 가격 | `!ㅇㄱㄹ` | ✅ 이전 완료 | 재료/요리 아이템 다중 가격 조회 |
| 젬 가격 | `!ㅈㄱㄱ` `!젬가격` | ✅ 이전 완료 | 경매장 젬 전체 가격 조회 |
| 보석 즉구 최저가 | `!보석가격` `!ㅂㅅ` `!ㅄ` | ✅ 이전 완료 | 겁화/작열 Lv7~10 즉구가 조회 |
| 거래소 가격 검색 | `!등급직업분류정렬가격 검색어` | ✅ 이전 완료 | 등급/직업/분류/정렬 조합 검색 |
