# API 명세

## 서버 엔드포인트 (Cloudflare Worker)

### 이미지 저장
- **POST** `/s`
- Body: `{ "image": "<base64>" }`
- Response: `<6자리 키>`
- 만료: 1시간 (KV TTL)

### 이미지 조회
- **GET** `/g/<key>`
- Response: `image/jpeg`

### OG 래퍼 (카카오링크용)
- **GET** `/e/<key>`
- Response: HTML (og:image 메타태그 포함)
- 카카오가 이 URL을 파싱하여 이미지 미리보기 생성

### 봇 메시지 수신
- **POST** `/bot`
- Header: `X-Bot-Secret: <BOT_SECRET>`
- Body:
  ```json
  {
    "room": "채팅방 이름",
    "msg": "메시지 내용",
    "sender": "발신자 이름",
    "isGroupChat": true
  }
  ```
- Response (텍스트 응답):
  ```json
  { "reply": "응답 텍스트" }
  ```
- Response (카카오링크):
  ```json
  { "reply": null, "kakaolink": { ... } }
  ```
- Response (처리 없음): `{ "reply": null }`
- 인증 실패: `401 { "error": "Unauthorized" }`
- 서버 오류: `500 { "reply": "삐빅! 서버 에러: ..." }`

### 카카오 세션 업로드
- **POST** `/session`
- Header: `X-Bot-Secret: <BOT_SECRET>`
- Body: `{ "cookies": { ... } }`
- Response: `{ "ok": true }`
- 용도: 모바일 로고봇에서 카카오 세션 쿠키를 서버 KV에 저장
- 만료: 30일 (KV TTL)

## 봇 명령어 목록

### 로스트아크

| 명령어 | 설명 |
|--------|------|
| `!정보 <이름>` | 캐릭터 전체 정보 (카카오링크) |
| `!배럭 <이름>` / `!ㅂㄹ` | 배럭 정보 |
| `!스킬 <이름>` | 스킬 정보 |
| `!보석 <이름>` | 보석 정보 (등급순 정렬, `레벨 이름 스킬명` 형식) |
| `!장비 <이름>` | 장비 정보 (`부위: 강화수치[상급재련] 세트명(품질)` 형식) |
| `!악세 <이름>` | 악세서리 정보 (`(품질) 깨:포인트` + 연마효과 `[상/중/하]` 등급, 팔찌 효과 포함) |
| `!앜패 <이름>` / `!ㅇㅍ` | 아크패시브 진화/깨달음/도약 효과 |
| `!아크 <이름>` / `!ㅇㅋ` | 아크그리드 코어/젬 정보 |
| `!낙원 <이름>` / `!ㄴㅇ` | 낙원력 |
| `!각인 <이름>` | 각인 정보 |
| `!내실 <이름>` | 내실 정보 |
| `!아바타 <이름>` | 아바타 정보 |
| `!즐 <이름>` / `!ㅈ` | 즐로아 검색 |
| `!로펙 <이름>` / `!ㄹㅍ` | 로펙 검색 (`https://lopec.kr/character/specPoint/<이름>`) |
| `!시너지` / `!ㅅㄵ` | 직업별 시너지 목록 |
| `!시간표` / `!ㅅㄱㅍ` | 로아 주간 일정 |
| `!각인도감` | 각인도감 스프레드시트 링크 |
| `!공홈` / `!ㄱㅎ` | 공식 홈페이지 |

### 경매/거래

| 명령어 | 설명 |
|--------|------|
| `!유각` / `!ㅇㄱ` | 유물 각인 검색 |
| `!ㅇㄱㄹ` | 에기르 재료 가격 (장인 1·2단계, 업화 [15-18] 제외) |
| `!젬가격` / `!ㅈㄱㄱ` | 아크그리드 젬 가격 (고급 등급 제외, 영웅/희귀만 출력) |
| `!클골` / `!ㅋㄱ` | 클리어 골드 |
| `!섬` / `!ㅅ` | 섬 정보 |
| `!ㅂㅂㄱ <금액>` | 파티 분배금 계산 (4/8/16인) |
| `!<등급><직업><분류><정렬> <검색어>` | 거래소 검색 |
| `!가격설명서` | 거래소 검색 파라미터 설명 |

| 명령어 | 설명 |
|--------|------|

## 외부 API

### 로스트아크 API
- 기본 URL: `https://developer-lostark.game.onstove.com`
- 인증: `Authorization: bearer <LOSTARK_API_KEY>`
- 주요 엔드포인트:
  - `GET /armories/characters/{name}` — 캐릭터 장비/빌드 전체 정보
  - `GET /characters/{name}/siblings` — 보유 캐릭터 목록

### 카카오 인증
- 로그인: `POST https://accounts.kakao.com/api/v2/login/authenticate.json`
  - 비밀번호는 AES 암호화 후 전송 (키: 로그인 페이지 `__NEXT_DATA__`에서 추출)
  - 세션 쿠키를 KV에 저장 (`session:kakao`, TTL 30일)
- 카카오링크 전송: `POST https://sharer.kakao.com/picker/link` → `POST /picker/send`
  - 앱 키: `KAKAO_API_KEY` (하드코딩, 공개키)
  - 채팅방 이름으로 대상 방 검색 후 전송
