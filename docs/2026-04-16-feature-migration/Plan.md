# Feature Migration & Cleanup

- **날짜**: 2026-04-16
- **담당**: developer / qa

## 목표
> msgbot/Bots 내 기능들을 server/src로 이전하고, 폐기 확정된 코드를 제거한다.

- `msgbot/Bots/크롤링/크롤링.js` — 이전 대상 3개 기능

## 이슈 및 메모
- 거래소/경매장 API는 POST 방식 → lostark-api.js에 `lostarkPost` 함수 추가 필요
- 아바타 이미지 전송은 Cloudflare Worker `/s` 엔드포인트 경유 (기존 크롤링 방식 유지)
- `!클골`, `!연마`는 Worker 정적 엔드포인트(`/e/CLEARGOLD`, `/e/POLISHINGACCESSORIES`) 링크 반환
- 낙원력은 원정대 형제 API(`/characters/{name}/siblings`) 기반
- 쌀 섬 일정은 게임콘텐츠 API(`/gamecontents/calendar`) 기반
